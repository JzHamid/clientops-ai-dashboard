# ClientOps AI Dashboard - Agent Instructions

This is a portfolio project for Jazhem Hamid.

## Goal
Build a polished SaaS-style admin dashboard that showcases software development, admin systems, AI automation readiness, and clean UI work.

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock data first
- Supabase and AI integrations later

## Rules
- Keep the code clean and beginner-readable.
- Use reusable components.
- Do not add unnecessary dependencies.
- Do not add authentication yet.
- Do not connect Supabase yet.
- Do not add real API keys.
- Do not expose secrets on the client.
- Use mock data in a separate file.
- Keep the design dark, modern, responsive, and portfolio-ready.
- Run npm run build after major edits.
